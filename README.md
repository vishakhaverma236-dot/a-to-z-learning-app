A to Z Learning App

An interactive Android educational application designed to help beginners learn A–Z alphabets with sound pronunciation and positive feedback.

Tools Used
- Android Studio
- Java

Features
- Alphabet learning from A to Z
- Clear sound pronunciation for each letter
- Interactive user interface
- “Good Job” pop-up messages for motivation
- Child-friendly design and navigation

Purpose
This project focuses on early learning education by combining audio-based pronunciation with interactive feedback to improve engagement and retention.
